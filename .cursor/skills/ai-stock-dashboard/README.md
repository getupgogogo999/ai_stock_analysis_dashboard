# Cursor Agent Skill：AI 股票分析台

本目录为 [Cursor Agent Skills](https://cursor.com/docs) 项目技能，帮助 AI 理解并维护本仓库。

## 文件

| 文件 | 说明 |
|------|------|
| [SKILL.md](./SKILL.md) | 主技能（架构、文件地图、改代码约定） |
| [reference.md](./reference.md) | 环境变量、脚本、API 参考 |

## 使用方式

1. 用 Cursor 打开本仓库（Project Skill 自动生效）
2. 或在对话中说明「按 AI 股票分析台项目 skill 来做」
3. 复制到 `~/.cursor/skills/ai-stock-dashboard/` 可在任意项目引用
